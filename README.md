# 📖 Overview

This repository contains my work for **Task 1 – Fundamentals of VLSI**. The objective of this task is to understand the fundamentals of Very Large Scale Integration (VLSI), digital logic design, and the complete VLSI design flow before moving on to RTL Design and Verilog HDL.

---

# 🎯 Key Outcomes

✔️ Explain what **VLSI** is and its importance.

✔️ Identify where VLSI is used in real-world applications.

✔️ Describe the complete **VLSI Design Flow**.

✔️ Understand the fundamentals of **Digital Logic Design**.

✔️ Design and simulate basic logic circuits.

✔️ Build a strong foundation for **RTL Design**.

✔️ Prepare for **Verilog HDL Programming**.

---

# 📚 Topics Covered

- Introduction to VLSI
- Applications of VLSI

- Complete VLSI Design Flow

- Verilog Basics
